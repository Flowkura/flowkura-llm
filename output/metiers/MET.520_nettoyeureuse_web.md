# nettoyeur/euse web
**ID:** MET.520

**Appellations:**
- Féminin: nettoyeuse web
- Masculin: nettoyeur web

## Autres appellations

### web cleaner
- Féminin: web cleaner
- Masculin: web cleaner

## Présentation

Préserver la réputation numérique d'une marque de luxe, d'une banque, d'une personnalité du showbiz ou du cinéma, d'un influenceur... telle est la mission des nettoyeurs web. Modérer, modifier et supprimer des contenus susceptibles de nuire à l'image d'une entreprise ou d'une personne permet aussi un meilleur référencement par les moteurs de recherche.

## En bref

Grands groupes internationaux, réseaux sociaux, entreprises marchandes, célébrités, politiques... tout le monde tient à préserver sa réputation numérique. Chargés de lire et de visionner des sites, des réseaux sociaux, ou tout emplacement sur Internet, les nettoyeurs web travaillent dans l'ombre pour modérer ou supprimer des contenus pouvant nuire à la e-réputation de leurs clients, car le Net est capable de véhiculer le pire comme le meilleur. Après un diagnostic des contenus, ils proposent à leurs clients une stratégie de communication pour améliorer leur image. Traquant tout ce qui se dit sur ces derniers, ils n'hésitent pas à remplacer certains commentaires par d'autres, plus élogieux, afin d'être mieux référencés par les moteurs de recherche. Bons communicants, férus de nouvelles technologies, les nettoyeurs web maîtrisent la création de contenus, la négociation avec les différents intervenants, l'animation de réseaux...
