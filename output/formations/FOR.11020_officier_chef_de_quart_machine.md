# officier chef de quart machine

## Informations générales

**Sigle:** OCQM
**Durée:** 3 ans
**Niveau de certification:** niveau 6
**Type:** formation d'école spécialisée

**Lien:** [https://www.francecompetences.fr/recherche/rncp/39798/](https://www.francecompetences.fr/recherche/rncp/39798/)

## Poursuite d'études

Les diplômes d'officier chef de quart machine et le diplôme de chef mécanicien 8000 KW sont délivrés après l'acquisition de la totalité des modules de formation

                - 
                    Le Brevet d'officier chef de quart machine après douze mois dont six de navigation comme élève dans le service machine et six mois de formation aux techniques d'atelier.

                - 
                    Le Brevet de second mécanicien 8000 kW est délivré après douze mois de navigation comme officier breveté dans le service machine.

                - 
                    Le Brevet de chef mécanicien 8000 kW est délivrée après 36 mois de navigation comme officier breveté dans le service machine ou 24 mois seulement si 12 ont été effectués comme second mécanicien.

## Métiers associés

- mécanicien/ne marine navigant
- mécanicien/ne bateaux
