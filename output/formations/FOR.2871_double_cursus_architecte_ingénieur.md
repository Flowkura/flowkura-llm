# double cursus architecte-ingénieur

## Informations générales

**Durée:** 7 ans
**Niveau de certification:** niveau 7
**Type:** diplôme des écoles d'architecture

## Description

Le double cursus architecte-ingénieur, est une formation post-bac, préparée par une école nationale supérieure d'architecte (ENSA) et une ou des écoles d'ingénieur associées. 

            La durée du parcours est de 7 ans. L'organisation des études est variable selon les établissements. Les élèves architectes suivent un enseignement scientifique (mathématiquen physique..) pendant leur formation. A l'issue des 3 années du double diplôme, les élèves obtiennent le Diplôme d'Etat d'architecte. Puis ils suivent deux années supplémentaires correspondant aux deux dernières années de spécialisation du cycle ingénieur pour obtenir le diplôme de l'école d'ingénieur.

## Métiers associés

- architecte-ingénieur/e
